List Style Type 

List Style Type is used to decide the type of List you want.

There are two classifications of lists:

1) Unordered List (ul)
Their default sign is disc.
Other types include circle, square etc.

2) Ordered List (ol)
Their default sign is natural numbers 1-n.
Other are:
a-z
A-Z
i
I

Note: Ordered List type can also be changed by "type" attribute as can be seen in my HTML Repository at GitHub.

Unordered List Default

  • One
  • Two
  • Three

Unordered List Customized

  • Three
