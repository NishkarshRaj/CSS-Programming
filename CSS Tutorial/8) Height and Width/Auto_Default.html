Automatic Setup of Height and Width => Default Setting. 

The height and width properties are used to set the height and width of an element.

The height and width can be set to auto (this is default. 
Means that the browser calculates the height and width), or be specified in length values, like px, cm, etc., or in percent (%) of the containing block.
Note: 
The height and width properties do not include padding, borders, or margins; they set the height/width of the area inside the padding, border, and margin of the element!
